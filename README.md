#Workplace Wellness Risk Monitor

#Problem Statement
Analyze employee health data to find employees at risk and recommend wellness actions.

#Features
- Detects anomalies using Z-score
- Groups employees into Low, Medium, High risk
- Suggests wellness actions for each risk group
- Shows summary reports and bar chart

#Skills Demonstrated
- AI/ML: Clustering (KMeans), Anomaly Detection
- Python Programming with Pandas, Scikit-learn
- Data Analysis and Trend Detection
- Clean, Modular Code Structure

#Files Included
- `Untitled6.ipynb`: Main Google Colab Notebook with full code
- `README.md`: Project documentation

#Output Example

| EmployeeID | Risk Group | Suggested Intervention |
|------------|------------|------------------------|
| 1          | Medium     | Encourage fitness, monitor heart rate |
| 5          | High       | Health checkup, reduce work hours, improve sleep |
| 7          | Low        | Maintain routine, encourage healthy habits |
